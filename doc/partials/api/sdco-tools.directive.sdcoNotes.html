sdcoNotes
directive in module sdco-tools

Description

Allow to add a note in your views. This note also contain an editable area to let users to add comments and save them. You can add several note in a view, but remember that the note is related to the view where you use it. For global note, have a look at sdcoNotesExport.

Usage

as element:
<sdco-notes>
</sdco-notes>

Directive info

  • This directive creates new scope.
