sdcoEditorTab
directive in module sdco-tools

Description

Define the content of a tab for its sdcoEditor directive parent.

Usage

as element:
<sdco-editor-tab
       type="{String}"
       heading="{String}">
</sdco-editor-tab>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
typeString

The type of the file: available values are: html, javascript, css. HTML content has to be escaped

headingString

the title of the tab

Example

<sdco-editor compile="false" compile-on-demand="true" js-fiddle="true">
 <sdo-editor-tab type="html" heading="index.html">
  <p> id="elt" my escaped html content </p>
 </sdco-editor-tab>
 <sdo-editor-tab type="javascript" heading="main.js">
  document.getElementById('elt').innerText='changed';
 </sdco-editor-tab> 
 <sdo-editor-tab type="css" heading="main.css">
  p{
  color: red;
  }
 </sdco-editor-tab>
</sdco-editor>
