sdcoEditorService
service in module sdco-tools

Description

This service is used internally to help sdo-editor directive behavior

This service shouldn't be used out of this module

Methods

  • getInstalledEditors()

    Get installed editors (in the current view)

    Returns
    Array

    an array of editor instances installed in the current view

  • installEditor(theElement, content, type, id, readOnly)

    Store an editor instance and to install it on the specified element with the specified content.

    Parameters
    ParamTypeDetails
    theElementElement

    the element to install the editor to

    contentString

    the content to display in the editor

    typeString

    the content type, which has to be one of javascript, html, css

    idString

    the id of the editor. Typically, its indice in the page, which is 1 if it appears first in the page, 2 if its the second, ... Information needed to store and retrieve its content based on the view when needed.

    readOnlyBoolean

    specifies if the editor should be in read-only mode

  • removeEditor(editor)

    Remove an installed editor instance (both in the DOM and in the array referencing all instances)

    Parameters
    ParamTypeDetails
    editorObject

    the editor instance to remove

  • run()

    Get the content of installed editors aggregated by type

    Returns
    Object

    an object containing one field by editor type, whose value is the aggregation of editors contents of this type. 

    {javascript:'all js content', css:'all css content', xml: 'all xml content'}
