menuSettings
directive in module sdco-tools

Description

Display a basic menu list, and allows to define an action when an option is selected/deselected

Usage

as element:
<menu-settings
       [settings-content="{array}"]>
</menu-settings>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
settingsContent
(optional)
array

angular expression evaluating to an array defining the menu. The array must contain well formed objects. Each object must contain the fields

  • label: The label displayed in the menu
  • selected: A boolean to specify that the element is selected
  • select: A function to execute when the menu is selected
  • deselect: A function to execute when the menu is deselected
