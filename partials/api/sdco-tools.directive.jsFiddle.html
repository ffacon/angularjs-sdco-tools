jsFiddle
directive in module sdco-tools

Description

Add an html link which allows to open a new jsFiddle

Usage

as element:
<js-fiddle
       fwk="{String}"
       version="{String}"
       title="{String}"
       description="{string}"
       [data="{array}"]>
</js-fiddle>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
fwkString

the name of the framework to use ( go here to get available values)

versionString

the version of the framework ( go here to get available values)

titleString

for this jsFiddle

descriptionstring

for this jsFiddle

data
(optional)
array

an array containing 3 string fields to be used for this jsFiddle: {html:'', css:'', js:'' }
